# First External Operator Outreach Kit v0.1

**Status:** Active Outreach Resource  
**Target:** Audit Firms, Compliance Providers, Certification Bodies

---

## 1. Purpose
This kit is designed to facilitate the onboarding of the first independent nodes into the DeciRepo verification network. It provides the technical, operational, and economic rationale for becoming a **Mirror Verifier**.

---

## 2. Why Become a Mirror Verifier?
- **Independent Verification Role:** Establish your organization as a neutral verifier of institutional decisions.
- **Early Network Position:** Gain priority status in the emerging verification graph.
- **Assurance Services:** Create a new revenue stream by offering certified, high-assurance rebuild services to decision issuers.
- **Technological Leadership:** Leverage DLX-based deterministic recomputation to automate complex audit tasks.

---

## 3. Implementation Path
To join the network, a candidate operator should follow these documents:
1. **[Deployment Guide](./REFERENCE_NODE_DEPLOYMENT_GUIDE_V0_1.md):** How to install and configure a node.
2. **[External Operator Program](./FIRST_EXTERNAL_OPERATOR_PROGRAM_V0_1.md):** Formal requirements and roles.
3. **[Economics Model](./DECIREPO_NETWORK_ECONOMICS_V0_1.md):** Understanding the publication/verification cost surface.

---

## 4. Pilot Expectations (Day Zero)
- **Uptime:** ≥ 30 days of consistent API availability.
- **Activity:** ≥ 10 successful cross-node verification events.
- **Identity:** Publicly resolvable node manifest and identity key.
- **Conformance:** 100% pass rate on the `dlx-ref` conformance suite.

---

## 5. Summary
Becoming a DeciRepo operator means moving from "trusting a report" to "running a proof." Join the network to provide the independent verification capacity that modern institutional decision-making requires.
